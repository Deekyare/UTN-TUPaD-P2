Analiza el siguiente código y responde: ¿Cuáles son expresiones y cuáles son
instrucciones? Explica la diferencia en un breve párrafo.

int x = 10; // Línea 1: Expresión

x = x + 5; // Línea 2: Expresión

System.out.println(x); // Línea 3: Instruccion

Diferencia entre expresiones e instrucciones:
Una expresión es una combinación de variables, literales, operadores y/o llamadas a métodos que se evalúa para producir un valor único (por ejemplo, x + 5 o 10). Las expresiones generan valores.
Una instrucción, en cambio, es una unidad completa de ejecución en un programa que realiza una acción, como declarar una variable, asignar un valor o invocar un método. Las expresiones suelen formar parte de instrucciones,
pero no todas las instrucciones contienen expresiones. Las instrucciones ejecutan acciones.
